**Searching and basic utilities**
```
- <Leader>sf - [S]earch [F]iles
- <Leader>sh - [S]earch [H]elp
- <Leader>fm - [F]ile [M]anager
- <Leader>sk - [S]earch [K]eymaps
- <Leader>ss - [S]earch [S]elect Telescope
- <Leader>s. - [S]earch Recent Files
- <Leader>/ -  Search using Fuzzy Finder in current buffer
- <Leader>s/ - Live Grep in Open Files
- <Leader>sn - Search Neovim files
- <Leader><Leader> - Find existing buffers
- <Leader>sr - [S]earch [R]esume
```
**Navigation and selection**
```
- <C> + (np) - Next/Previous cmp-completion item
- <C> + y - Confirm cmp-completion item
- <C> + b - Scroll documentation window [b]ack
- <C> + f - Scroll documentation window [f]orward
- <C> + <Space> - Manually trigger cmp-completion
- <C> + l - LuaSnip expand or jump
- <C> + h - LuaSnip jump
- va( ')' '}' ']' ) 
- yinq - Yank Inside Next Quote
- ci' - Change Inside Quote
- sd' - [S]urround [D]elete Quotes
```
